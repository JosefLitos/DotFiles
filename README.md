# GNU/LINUX Dotfiles

In this repository are my personal dotfiles which I use on my arch system.
I include source code for my terminal of choice: st, as well as used themes for gtk

# Dotfiles in this repo
+ i3
+ i3blocks
+ nvim
+ rofi
+ bash
+ fish
+ vlc
+ ranger
+ picom
+ termite
+ htop
